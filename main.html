"2. Algorithms for Nonconvex Compressive Sensing"

Code for the paper's main algorithm is given here for both Octave and Python.

Summary

Essentially we iteratively solve a constrained linear least squares (LS) problem for which the weights for the present step, $w_i$, depend on the solution from previous step, $u_i^{(n-1)}$: $$w_i=\left( \left(u_i^{(n-1)}\right)^2+\epsilon \right)^{p/2-1} \tag{6}$$ For the LS solution, let $Q_n=\mathrm{diag}(1/w_i)$, so that $$u^{(n)}=Q_n \Phi^T \left( \Phi Q_n \Phi^T \right)^{-1}b \tag{5}$$ The iteration is initialized with $$\Phi u_0 = b$$.
